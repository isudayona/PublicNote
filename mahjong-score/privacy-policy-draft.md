## 2. 取得する個人情報の項目

本アプリは、以下の個人情報を取得します。

| 種別 | 項目 | 取得方法 |
|---|---|---|
| 識別子 | Adapty 匿名ユーザー ID | サブスクリプション管理目的で Adapty SDK が自動生成 |
| 識別子 | 広告識別子（IDFA） | iOS の App Tracking Transparency に基づくユーザー同意取得時のみ |
| 購入情報 | サブスクリプション購入履歴 | App Store の決済処理経由で Adapty が記録 |
| 使用状況データ | アプリ利用に関する統計情報 | Adapty SDK / Google AdMob SDK が匿名で収集 |
| 写真・動画 | OCR 用カメラ画像 | OCR 機能利用時にユーザーが任意でカメラ画像を提供（当方サーバーへの送信なし） |

## 3. 利用目的

取得した個人情報は、以下の目的で利用します。

- サブスクリプションの提供・管理（Adapty 経由）
- 広告配信および広告効果の測定（Google AdMob 経由）
- 本アプリの機能提供・改善
- 不正利用の防止
- ユーザーからのお問い合わせへの対応

## 4. 第三者提供

当方は、ユーザーの同意がある場合または法令で認められる場合を除き、ユーザーの個人情報を第三者に提供しません。

## 6. 安全管理措置

当方は、取得した個人情報の漏えい、滅失またはき損の防止その他個人情報の安全管理のため、以下の措置を講じます。

- **技術的安全管理措置**: 本アプリのデータは原則として端末内（SQLite データベース）に保存され、外部サーバーへ送信されません。iOS のデータ保護機能（Data Protection）に基づく端末内暗号化に依拠します。
- **委託先の管理**: 委託先（Adapty・Google）は各社のプライバシーポリシーに基づく安全管理措置を講じています。

ユーザーは、自己の個人情報について、以下の請求を行うことができます。

- 利用目的の通知
- 開示
- 訂正、追加または削除
- 利用の停止または消去
- 第三者への提供の停止

請求は、下記「個人情報に関する苦情・お問い合わせ窓口」までご連絡ください。当方は、本人確認を行ったうえで、合理的な期間内に対応します。手数料は無料です。

なお、本アプリで取得・保管する識別子情報は、ユーザー個人を特定可能な情報を含まないため、特定のユーザー識別子に関する開示請求についてはお応えできない場合があります。
